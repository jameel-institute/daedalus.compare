## Output helper functions

Get output for comparison from multiple scenarios’ output.

- [`get_cost_data()`](https://jameel-institute.github.io/daedalus.compare/reference/get_cost_data.md)
  : Get cost data from DAEDALUS scenarios
- [`get_costs_list()`](https://jameel-institute.github.io/daedalus.compare/reference/get_costs_list.md)
  : Get pandemic costs from a list of model outputs
- [`get_econ_cost_data()`](https://jameel-institute.github.io/daedalus.compare/reference/get_econ_cost_data.md)
  : Get economic cost details from an output data.table
- [`get_econ_costs_list()`](https://jameel-institute.github.io/daedalus.compare/reference/get_econ_costs_list.md)
  : Get details of economic costs from model outputs
- [`get_epicurve_data()`](https://jameel-institute.github.io/daedalus.compare/reference/get_epicurve_data.md)
  : Get epidemiological curves from model data
- [`get_epidata_list()`](https://jameel-institute.github.io/daedalus.compare/reference/get_epidata_list.md)
  : Get incidence data from a list of model outputs
- [`get_summary_data()`](https://jameel-institute.github.io/daedalus.compare/reference/get_summary_data.md)
  : Get summary data from DAEDALUS scenarios
- [`get_summary_list()`](https://jameel-institute.github.io/daedalus.compare/reference/get_summary_list.md)
  : Get epidemic summary data from a list of model outputs
